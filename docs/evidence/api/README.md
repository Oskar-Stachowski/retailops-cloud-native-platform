# API Evidence

Last reviewed: 2026-05-12

This folder stores curated API evidence for recruiter and technical review.

| File | What it proves | Related checklist row | Validation note |
|---|---|---|---|
| `startup-log.md` | FastAPI application starts under Uvicorn and responds to `/health`. | API-001 | Captured from a local run on `127.0.0.1:8011`. |
| `openapi-snapshot.json` | `/openapi.json` is generated by the running FastAPI app and contains the RetailOps API schema. | API-009 | Captured with `curl` from the running API and formatted with `jq`. |

## Refresh Commands

```bash
cd services/api
PYTHONPATH=. .venv/bin/uvicorn app.main:app --host 127.0.0.1 --port 8011
```

In another terminal:

```bash
curl -fsS http://127.0.0.1:8011/health
curl -fsS http://127.0.0.1:8011/openapi.json -o docs/evidence/api/openapi-snapshot.json
jq . docs/evidence/api/openapi-snapshot.json > /tmp/retailops-openapi-snapshot.json
mv /tmp/retailops-openapi-snapshot.json docs/evidence/api/openapi-snapshot.json
```
